DOCUMENTATION


Offline Payment (v2.0) plugin for LaraClassified and JobClass.

Installation

To install this plugin:
  • Copy the plugin's (uncompressed) folder into the script's plugins folder like this: /extras/plugins/offlinepayment/
  • Go to your Admin panel → Plugins (and click on "Install") to install it.

Compatibility

  • v2.0+ => LaraClassified v7.1+ and JobClass v5.9+
  • v1.9 => LaraClassified (v6.9 to v7.0.x) and JobClass (v5.7 to v5.8.x)
  • v1.8 => LaraClassified (v6.6 to v6.8) and JobClass (v5.4 to v5.6)
  • v1.7 => LaraClassified (v6.1 to v6.5) and JobClass (v4.9 to v5.3)
  • v1.6 => LaraClassified (v5.7 to v6.0) and JobClass (v4.5 to v4.8)
  • v1.5 => LaraClassified (v5.2 to v5.6) and JobClass (v4.0 to v4.4)
  • v1.4 => LaraClassified v5.1 and JobClass v3.9
  • v1.3 => LaraClassified (v4.9 to v5.0) and JobClass (v3.7 to v3.8)
  • v1.2 => LaraClassified (v4.6 to v4.8) and JobClass (v3.4 to v3.6)
  • v1.0 to v1.1 => LaraClassified (v3.8 to v4.5) and JobClass (v2.6 to v3.3)

Update

Replace all the plugin's files on your server.

Configuration

  • Go to: Admin panel → Payment → Payment Method → (Edit the "Offline Payment" description), to add information about your various offline payment methods.

Usage

  • After receiving the amount of an offline payment, you must manually approve the payment in the Admin panel → Payments → List → (Search the "Reason of the payment" using the Ad ID and check the "Approved" checkbox)

Translation

To translate this plugin into your language:
  • Go to the languages folder at /extras/plugins/offlinepayment/resources/lang/
  • Duplicate the folder /extras/plugins/offlinepayment/resources/lang/en/ to /extras/plugins/offlinepayment/resources/lang/<abbr>/
    NOTE: Don't forget replace <abbr> by your language code (e.g. "de" for German language).
  • Open the files:
    /extras/plugins/offlinepayment/resources/lang/<abbr>/mail.php,
    /extras/plugins/offlinepayment/resources/lang/<abbr>/messages.php,
    /extras/plugins/offlinepayment/resources/lang/<abbr>/sms.php
    and translate them.
    NOTE: To translate the file, you must change the values and not the keys of PHP array.
  • If you want to translate the Offline Payment description DB entry, you can do it by adding the value of the 'payment_method_description' key in the plugin /extras/plugins/offlinepayment/resources/lang/<abbr>/messages.php language file.

Uninstallation

To uninstall this plugin, go to your Admin panel → Plugins (and click on "Uninstall") to uninstall it.

Removing

To remove this plugin:
  • Uninstall it
  • Go in the folder /extras/plugins/ and remove the folder offlinepayment/.
